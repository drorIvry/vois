# Security Policy

## Supported Versions

Only the latest release is supported with security updates.

## Reporting a Vulnerability

Please report vulnerabilities privately via
[GitHub Security Advisories](https://github.com/drorIvry/vois/security/advisories/new)
or by email to dror@qualifire.ai. Do not open public issues for security
reports.

You should receive a response within a few days. Please include reproduction
steps and the macOS/Vois versions affected.

## Scope notes

Vois runs fully offline (no network access at runtime) and requests the macOS
Accessibility permission to read the text you select. Reports about the
capture pipeline (selected-text handling, clipboard save/restore) and the
permission model are especially welcome.
